# ansible-youtube-dl

Install the latest stable youtube-dl via Homebrew.

## Dependencies

* [icopp.homebrew](https://github.com/icopp/ansible-homebrew)
* [icopp.python](https://github.com/icopp/ansible-python)

## Example Playbook

```
  - hosts: all
    roles:
      - role: icopp.youtube-dl
```
